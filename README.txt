Виселица (Hangman)

Консольная игра на C++. Компьютер загадывает слово, игрок угадывает буквы. За 6 ошибок — проигрыш.

Как запустить:

Требования
- Компилятор C++ (g++, clang++, MSVC)

Компиляция и запуск

Linux / macOS:
  bash
g++ hangman.cpp -o hangman
./hangman
Windows:
g++ hangman.cpp -o hangman.exe
hangman.exe

Правила игры

1. Компьютер загадывает слово (существительное на русском языке).
2. Игрок называет по одной букве.
3. Если буква есть в слове — она открывается.
4. Если буквы нет - засчитывается ошибка (всего 6 ошибок).
5. Игрок побеждает, если открыл все буквы до 6 ошибок.
6. После окончания игры можно начать новый раунд.
